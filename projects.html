Occam's Vault

Occam's Vault is a self-hosted password management solution designed to provide secure storage for sensative Information

  • Key Features:

    This aplication utilizes AES for data at rest and argon2ib for hashing the master password. No credentials are stored in plaintext. The application implements a secure login and registration process. It was imperitive for this project to remain privacy while having high availability. Therefore no data is ever transferred out to the external network. The application implements incremental, differential, and full abckup stragtegies to ensure data integrity and availability, while allowing users to export their database to other password managers, ensuring flexibility and user control

  • Technology Stack:
    • Backend Development: Python and the Bottle framework
    • Database: MySQL
    • Encryption: AES for data at rest, Argon2i for password hashing, SSL/TLS for network communication
    • Certification: Let's Encrypt for obtaining SSL/TLS certificates
